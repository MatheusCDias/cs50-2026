# CS50-2026: Introduction to Computer Science (Harvard)

This repository contains my studies, problem sets, and projects developed during **CS50x**, the introductory Computer Science course at Harvard University.

The course covers the fundamentals of programming, algorithms, and software development, using various languages ​​and technologies.

## Technologies and Concepts
* **Languages:** C, Python, SQL, HTML/CSS, JavaScript.
* **Algorithms:** Search (Linear, Binary), Sorting (Bubble, Selection, Merge).
* **Data Structures:** Arrays, Linked Lists, Hash Tables, Trees.
* **Topics:** Memory, Pointer Management, Databases, Security, and Web Development.

## How to run the projects
Most C programs were compiled in the **CS50 Codespace** environment.
To run locally, you will need the `cs50.h` library or adapt the code to the standard libraries.

```bash
# Example: Compiling a C program
make filename

# Running the executable
./filename
```
